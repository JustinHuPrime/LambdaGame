Lambda Calculus Syntax

Lambda Game uses a slightly modified version of lambda calculus syntax - instead of allowing function application using whitespace, we require parentheses for function application, like languages in the Lisp family.

Construct Description Syntax
Variables A variable is any string of non-whitespace characters excluding .\() name
Lambda functions Lambda functions here are always one-argument functions - they have an argument and a body, where the argument is some variable name, and the body is another expression. (\argument.body)
Function application To apply a function, surround the function expression and its argument expression in parentheses with whitespace in between them. (function argument)
